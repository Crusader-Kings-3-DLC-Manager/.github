# **CK3Unlocker: Automatic Crusader Kings 3 DLC Unlocker & Activator**

**CK3Unlocker** is an all-in one tool that automatically activates and unlocks DLCs in **Crusader Kings 3**. Note that you have to own a copy of the game in order to unlock DLCs.

## 🧰 **Dependencies:**

- A **Steam version** of **Victoria 3**
- The **latest release** of **CreamAPI**

## 🚀 **Activation Guide**

### Get CreamAPI and follow the instructions
- Download the latest release of **CreamAPI**.
- Extract the archive to any location (it’s fully portable).
- Launch **CreamAPI** and follow the setup instructions to activate DLCs for **Victoria 3**.

---

## 🔧 **Troubleshooting**

| Problem                        | Solution                                                                 |
|---------------------------------|--------------------------------------------------------------------------|
| DLCs not showing in-game        | Ensure CreamAPI was run as Administrator and restart Steam.              |
| Game crashes after launch       | Use the “Uninstall” option in CreamAPI and reinstall unlockers.         |
| Missing DLL errors              | Reinstall Koaloader via CreamInstaller’s context menu.                   |
| Still doesn’t work              | Re-run CreamAPI and refresh Steam appinfo for Crusader Kings 3.    |

---

## 🧠 **Tips & Notes**

- **Back up your Crusader Kings 3 folder** before making changes.
- **SteamCMD** will download automatically if missing.
- Works with **official Steam builds only** (no cracked or repack versions).
- If **Crusader Kings 3** updates, simply rerun **CreamAPI** to reapply the unlockers.
